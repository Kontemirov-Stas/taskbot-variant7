## ⚙️ Реалізовані функції

### Обов'язкові вимоги ✅
| Вимога | Реалізація |
|--------|-----------|
| Streamlit як основний фреймворк UI | `app.py` — весь UI на Streamlit |
| Повноцінний чат-інтерфейс | `st.chat_message()` + `st.chat_input()` |
| Збереження історії через `st.session_state` | `messages`, `tasks`, `next_task_id` |
| Інтеграція LLM (OpenAI gpt-4o-mini) | `agent.py` → `create_graph()` |
| Стрімінг відповідей | `st.spinner()` під час генерації |
| LangGraph-агент (Варіант 7) | ReAct-архітектура, 5 інструментів |
| Sidebar з налаштуваннями | режим, температура, токени, фільтри |
| 2–3 спеціалізованих UI-компоненти | Kanban (columns), bar_chart, metric, dataframe, expander |
| Структура коду (модулі/функції/коментарі) | `app.py` + `agent.py`, коментарі скрізь |
| Тестування на сценаріях | 5 сценаріїв нижче |

### Додаткові можливості ✅
- 📊 **Вкладки** (`st.tabs()`) — 4 вкладки: Чат, Kanban, Статистика, Всі задачі
- 📥 **Завантаження файлів** — чат (JSON/TXT), задачі (JSON/CSV)
- 📈 **Візуалізація даних** — `st.bar_chart()` по статусах і пріоритетах
- 🎨 **Кастомна CSS-стилізація** — Kanban-картки, кольори, анімації
- 📝 **Редагування системного промпта** в бічній панелі

## 🛠️ Інструменти агента

| Інструмент | Опис |
|-----------|------|
| `create_task(title)` | Створення нової задачі |
| `set_task_done(task_id)` | Позначення задачі виконаною |
| `set_task_in_progress(task_id)` | Переведення в статус "в процесі" |
| `list_open_tasks()` | Список відкритих задач |
| `daily_summary()` | Щоденний підсумок |

## 🧪 Приклади діалогів (5 сценаріїв)

### Сценарій 1 — Звичайний чат (додавання задач)
```
Ви: Додай задачу: написати звіт до п'ятниці
TaskBot: ✅ Створено задачу: 'написати звіт до п'ятниці' (id=1)

Ви: Ще одну: підготувати презентацію для менеджера
TaskBot: ✅ Створено задачу: 'підготувати презентацію для менеджера' (id=2)
```

### Сценарій 2 — Використання інструментів агента
```
Ви: Що в мене ще не зроблено?
TaskBot: 📋 Відкриті задачі:
  1. [open] написати звіт до п'ятниці — 🟡 Середній
  2. [open] підготувати презентацію для менеджера — 🟡 Середній
```

### Сценарій 3 — Зміна статусу
```
Ви: Познач задачу 1 як виконану
TaskBot: ✅ Позначив задачу 1 ('написати звіт до п'ятниці') як виконану.

Ви: Почав задачу 2
TaskBot: 🔄 Задача 2 ('підготувати презентацію для менеджера') переведена в статус 'в процесі'.
```

### Сценарій 4 — Підсумок дня
```
Ви: Підсумуй мої задачі на сьогодні
TaskBot: 📊 Підсумок задач:
  - 📋 Всього: 2
  - ✅ Виконано: 1
  - 🔄 В процесі: 1
  - 📝 Відкритих: 0
```

### Сценарій 5 — Демонстрація UI-компонентів (Kanban)
- Відкрити вкладку «📊 Kanban-дошка»
- Переміщати задачі кнопками ▶️ / ✅ / ↩️
- Змінювати пріоритети через selectbox
- Переглядати статистику у вкладці «📈 Статистика»
